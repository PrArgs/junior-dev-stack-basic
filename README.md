# JuniorZ Developer Bootcamp

A comprehensive end-to-end guide for junior software developers, covering the essential tools and practices needed to work effectively in a modern engineering team.

## About This Project

This static website serves as a practical learning resource for junior developers joining their first professional team. Rather than focusing on coding tutorials, it covers the **collaboration tools and workflows** that teams rely on daily but rarely teach explicitly.

## Topics Covered

### Git & GitHub
- Mental model for version control
- Core commands every developer needs
- Real-world workflow (branch → commit → PR → review → merge)
- Branching best practices
- Common problems and how to fix them
- Do's and Don'ts for team collaboration

### Jira
- Why teams use work tracking systems
- Scrum methodology basics
- Core concepts: tickets, boards, sprints
- Daily use cases for junior developers
- Linking code to Jira tickets
- Professional communication around blockers and priorities

### Slack
- Professional team communication vs personal messaging
- Slack compared to WhatsApp
- Channels, threads, and structured discussions
- Working with Slack and Jira together
- Professional communication expectations

## Features

- Clean, responsive design inspired by JuniorZ101
- Embedded YouTube tutorial videos
- Interactive code blocks with copy functionality
- Tip boxes, checklists, and comparison grids
- Mobile-friendly layout

## Tech Stack

- Pure HTML5 & CSS3
- No build tools or frameworks required
- Google Fonts (Raleway)
- GitHub Pages compatible

## Local Development

Simply open `index.html` in your browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

## Deployment

This site is designed to be hosted on GitHub Pages. Simply push to the `main` branch and enable GitHub Pages in your repository settings.
